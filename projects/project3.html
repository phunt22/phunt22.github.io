Ripple

React Native
App Store GitHub

Overview

Ripple is a student marketplace application designed for UW students to buy, sell, and trade items within the campus community. This project is being developed as a part of Dubhacks Next Batch 4, currently with 200+ users

Next Steps

Our team plans to continue working on this project, and plan to:

  1. Get more users on the platform
  2. Keep making improvements
  3. Develop features to increase user engagement
  4. Integrate a POS system (probably Stripe) and allow users to sell items as vendors––enabling student entrepreneurship
