Dr. Bayes: COVID-19 Symptom Analysis

JavaScript Data Science
Live Demo Repository

Overview

In this project for CSE 442 (Data Visualization), I wanted to visually demonstrate how Bayes Theorem works by showing how a patient's symptom profile changes what disease they are most likely to have using real data from the COVID pandemic.

While this project was designed for 3-4 students, I did it by myself due to health issues. Despite working alone, I not only met the project requirements but actually earned extra credit.
