KInARowBot

Python AI
View Game Script

Overview

KInARowBot is an AI-powered agent designed to play K-in-a-Row with Forbidden Squares, a strategic variation of tic-tac-toe, for CSE473 (Artificial Intelligence) at UW.

Assignment

The main challenges were creating an agent that could:

  • Play on variable-sized boards with configurable K-in-a-row win conditions
  • Handle forbidden squares and handicap positions
  • Implement efficient minimax search with alpha-beta pruning
  • Generate contextually appropriate dialogue while playing
  • Operate effectively under time constraints

Solution

I implemented several innovative features:

  • Efficient minimax algorithm with alpha-beta pruning for optimal move selection
  • Sophisticated static evaluation function considering elements such as an offensive and defensive heuristic
  • Dynamic depth adjustment based on time constraints
  • Implemented OpenAI's API to generate utterances for the agent as "LeBot James" (see the game script)
  • Support for any game configuration by using Q-Learning to generate a policy

Results

The KInARowBot achieved notable success:

  • Advanced to the semifinals of the class tournament (top 50% of competitors)
  • Never lost a match in tournament play (only tied in later rounds)
  • Earned extra credit for the LeBot James personality implementation
  • Demonstrated effective defensive strategy, though conservative play prevented advancement in later rounds

Reflections

While the agent performed exceptionally well defensively, the experience taught me valuable lessons about balance in AI strategy. A more aggressive approach might have yielded victories instead of ties in the later tournament rounds. Unfortunately, the original code was lost when my laptop was wiped after my phone was stolen so I will never be able to revisit it, but the project demonstrated my ability to implement effective game-playing algorithms and create engaging AI personalities.
